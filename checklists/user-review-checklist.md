# User Review Checklist

Use this checklist after Codex builds each major section.

## Foundation

- [ ] Login page exists.
- [ ] Dashboard opens after login.
- [ ] PWA manifest exists.
- [ ] README explains setup in plain language.

## Workout logger

- [ ] Timer ribbon appears after workout starts.
- [ ] Exercises can be selected.
- [ ] Exercises can be reordered.
- [ ] Exercise colors match muscle groups.
- [ ] Workout notes exist.
- [ ] Exercises/sets can collapse.
- [ ] Completed sets/exercises can be hidden.
- [ ] Completing exercise starts rest before next exercise.

## Program builder

- [ ] Days can be reordered.
- [ ] Day content can be edited.
- [ ] Exercises can be reordered within a day.

## Progression

- [ ] Recommendations are editable.
- [ ] Recommendations can be accepted as default.
- [ ] Recommendations can be log-only.
- [ ] Cycle progression can be skipped.
- [ ] Skipped recommendations still appear in log.

## Import

- [ ] Workbook import finds Log sheet.
- [ ] Import review appears before commit.
- [ ] Aliases can be reviewed.
- [ ] Bodyweight/calisthenics are excluded from resistance volume.
- [ ] Unilateral movements double volume.

## Analytics

- [ ] Cycle lookback can be changed.
- [ ] Exercise tracker exists.
- [ ] PRs appear.
- [ ] Progression log appears.

## Settings

- [ ] Theme/UI/button settings exist.
- [ ] Exercise database is grouped by muscle group.
- [ ] New exercise can be added.
