# Preview v2 changelog

## All screens

The prototype now includes a persistent timer ribbon once a workout is started. It shows session duration, rest status, the selected exercise, set progress, and shortcuts back to the logger.

## Workout

Implemented selectable exercise cards, drag-and-drop exercise ordering, muscle-group color themes, workout-specific notes, collapsible exercises, collapsible set details, hide completed sets, hide completed exercises, and a **Complete + rest** workflow. Completing an exercise marks sets complete, starts the configured rest timer, and selects the following exercise.

## Program

Implemented draggable program days, structured day editing, selectable program exercises, and drag-and-drop ordering of exercises within the selected day after recommendation.

## Progression

Added global **Skip progression for this cycle** behavior. Recommendations are still calculated and appended to a detailed progression log that analytics can consume. Recommendation rows are editable before being accepted as future defaults.

## Analytics

Added user-selected cycle lookback and a per-exercise tracker with history-style rows. Progression logs are visible in analytics.

## Settings

Added advanced configuration for theme mode, accent color, density, button size, font scale, button layout, persistent timer ribbon, timing/export defaults, and an editable exercise database grouped by Chest, Back, Shoulders, Biceps, Triceps, Core, and Legs.
